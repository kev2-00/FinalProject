# Simple Site Report

## Site Purpose

This website is an **Ethics & Law Quiz**. Its purpose is to help users practice scenario-based questions about:

- CFAA
- DMCA
- Responsible Disclosure & Privacy

The site is designed as a simple learning tool where users read a situation, choose an answer, and then review their score at the end.

## GitHub Pages Link

The site is available through GitHub Pages at:

[https://perzival-22.github.io/FinalProject/](https://perzival-22.github.io/FinalProject/)

## How the Site Works

GitHub Pages hosts the project as a static website, which means it serves the files in this repository directly in the browser.

When a visitor opens the link:

1. `index.html` loads the main page.
2. `css/style.css` controls the design and layout.
3. `js/data.js` provides the quiz questions and answers.
4. `js/quiz.js` runs the quiz logic, including topic selection, question flow, and scoring.
5. `js/results.js` shows the final results after the quiz is completed.

Because the project uses only HTML, CSS, and JavaScript, it does not need a backend server or database. This makes it easy to publish and access through GitHub Pages.
